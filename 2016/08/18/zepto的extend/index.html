Zepto的extend API

参数说明

  • source 需要被复制的对象，可以多个
  • target 为需要被返回的对象，可以为空对象，但不能是null,当target为空时，返回source
  • true 深拷贝

深拷贝

对象 {} ,数组[]: 递归复制,

注意：实际写深拷贝方法时要注意忽略掉一些浏览器原生对象，如：window，document之类的
